# Akshaya Cookie Company Automation

## Problem

Small businesses often manually collect customer orders, update spreadsheets, and send confirmation emails, which takes time and increases the chance of human error.

## Solution

Built a no-code automation workflow using Make.com to automate the complete order management process.

## Workflow

Google Forms → Google Sheets → Gmail

## What it does

* Captures customer order submissions automatically
* Stores all order details in Google Sheets
* Sends personalized confirmation emails instantly
* Reduces manual data entry
* Improves response time

## Tech Stack

* Make.com
* Google Forms
* Google Sheets
* Gmail

## Features

* Real-time form monitoring
* Automated data logging
* Email automation
* Filter-based workflow logic
